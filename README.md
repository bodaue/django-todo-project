### To Do App

<p> A simple ToDo app built with <a href="https://www.djangoproject.com/">Django</a></p>
 
---
</div>

### Features

- Create a user account
- Logout and login to your account
- Create your own tasks
- Edit tasks in the list
- Delete tasks from the list
- Mark tasks as complete
- Mark tasks as incomplete

### Tech Stack

- [Python v3.10.2](https://docs.python.org/3.10/)
- [Django v4.2.3](https://docs.djangoproject.com/en/4.2//)
